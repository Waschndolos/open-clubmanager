# open-clubmanager

# About Open-Clubmanager
Open-Clubmanager is a program which can be used by small clubs (Verein in German). It will be available for free and provide basic functionality to manage a club.

# Why?
Volunteeringly managing a club is a big task. You need to answer Mails, Mange the club list, Organize Events, and so on. All this can be made easier with software.
Unfortunately such software usually costs money. Money many clubs cannot affort. I'd like to give something back to the community of volunteer in writing this program.

# Want to contribute?
This is totally green field :) So I warmly welcome any Java developer who want's to join this project. 

# Rough design
* Spring boot
* Some UI - not yet specified
* Multiple DB's possible. From Text based DB on a Share to a mysql on some hosted environment. 
* Desktop Application - No Web! Download it, Install it, Run it. I don't want to provide Hosting.
